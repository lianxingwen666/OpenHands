## ⚠️ 重要声明

**本工具仅供学习和研究目的使用，请务必遵守以下原则：**

1. **合规使用**: 使用前请确保符合学校相关规定和政策
2. **真实评教**: 建议进行真实、客观、负责任的课程评教
3. **个人责任**: 使用者需对使用行为承担相应责任
4. **学习目的**: 本工具主要用于技术学习和自动化研究

## 功能特性

### 🚀 核心功能
- **一键填写**: 自动填写评教表单的单选项和文本评语
- **智能识别**: 自动识别页面中的评教元素
- **自定义配置**: 支持自定义评分等级和评语内容
- **预览模式**: 预览页面中的评教元素
- **快捷键支持**: Ctrl+Shift+E 快速填写

### 🎯 技术特点
- **浏览器扩展**: 基于Chrome扩展技术
- **无需安装**: 直接在浏览器中运行
- **安全可靠**: 所有操作在本地执行
- **用户友好**: 简洁直观的操作界面

## 安装方法

### 方法1: 开发者模式安装（推荐）

1. **下载扩展文件**
   ```bash
   # 下载整个 auto_evaluation_extension 文件夹
   ```

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入: `chrome://extensions/`
   - 或者: 菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `auto_evaluation_extension` 文件夹
   - 点击"选择文件夹"

5. **确认安装**
   - 扩展应该出现在扩展列表中
   - 浏览器工具栏会显示扩展图标

### 方法2: Python脚本方式

如果您更喜欢使用Python脚本：

1. **安装依赖**
   ```bash
   pip install selenium beautifulsoup4 requests
   ```

2. **下载ChromeDriver**
   - 访问: https://chromedriver.chromium.org/
   - 下载对应Chrome版本的驱动
   - 将驱动添加到系统PATH

3. **运行脚本**
   ```bash
   python auto_evaluation_tool.py
   ```

## 使用方法

### 浏览器扩展使用

1. **访问评教页面**
   - 登录学校评教系统
   - 进入具体的评教页面

2. **打开扩展**
   - 点击浏览器工具栏中的扩展图标
   - 或使用快捷键 `Ctrl+Shift+E`

3. **配置参数**
   - **评分等级**: 选择1-5的评分（5为最高）
   - **评语内容**: 输入或修改评语文本
   - **操作延时**: 设置操作间隔时间（秒）

4. **执行操作**
   - **预览**: 查看页面中的评教元素
   - **自动填写**: 执行自动填写操作

5. **确认提交**
   - 检查填写结果
   - 手动点击提交按钮

### 快捷键操作

- `Ctrl+Shift+E`: 使用默认配置快速填写

## 配置说明

### 评分等级
- **5**: 非常满意（推荐）
- **4**: 满意
- **3**: 一般
- **2**: 不满意
- **1**: 非常不满意

### 评语模板
```
默认评语: "老师教学认真负责，课程内容丰富，受益匪浅。"

其他建议:
- "课程内容充实，教学方法得当，收获很大。"
- "老师讲解清晰，课堂氛围良好，学到了很多知识。"
- "教学水平高，课程设计合理，非常有帮助。"
```

### 延时设置
- **0.5-1秒**: 快速模式
- **1-2秒**: 标准模式（推荐）
- **2-5秒**: 谨慎模式

## 技术原理

### 扩展架构
```
manifest.json     # 扩展配置文件
popup.html/js     # 用户界面
content.js        # 页面内容脚本
styles.css        # 样式文件
```

### 工作流程
1. **页面检测**: 识别评教页面和表单元素
2. **元素定位**: 查找单选按钮和文本框
3. **自动填写**: 模拟用户操作填写表单
4. **视觉反馈**: 提供操作状态和结果提示

### 安全机制
- **本地执行**: 所有操作在浏览器本地执行
- **无数据传输**: 不向外部服务器发送数据
- **用户控制**: 用户完全控制填写内容和时机

## 故障排除

### 常见问题

1. **扩展无法加载**
   - 检查是否启用了开发者模式
   - 确认文件夹结构完整
   - 重新加载扩展

2. **无法识别评教页面**
   - 确认在正确的评教页面
   - 检查页面是否完全加载
   - 尝试刷新页面

3. **自动填写失败**
   - 检查页面元素是否发生变化
   - 尝试增加操作延时
   - 手动预览检查元素

4. **快捷键不工作**
   - 确认页面获得了焦点
   - 检查是否有其他扩展冲突
   - 尝试重新加载页面

### 调试方法

1. **打开开发者工具**
   - 按F12或右键→检查
   - 查看Console标签页的日志

2. **检查扩展状态**
   - 访问 `chrome://extensions/`
   - 查看扩展是否正常运行

3. **查看错误信息**
   - 点击扩展详情
   - 查看"错误"部分的信息
