## 🎲 智能评分原理

### 为什么不能都是6分？
1. **系统检测**：相同分数容易被系统标记为异常
2. **数据分析**：真实评教数据通常有分布差异
3. **可信度**：多样化的评分更符合真实情况

### 智能算法
```javascript
// 加权随机选择算法
function getSmartScore(strategy) {
    const weights = strategies[strategy].weights;
    const random = Math.random();
    let cumulative = 0;

    for (score in weights) {
        cumulative += weights[score];
        if (random <= cumulative) {
            return score;
        }
    }
}
```

   - 检查开发者模式是否启用

   - 增加操作延时
   - 使用预览功能检查元素
   - 查看浏览器控制台错误
