我为您创建了两种自动评教解决方案，可以帮助您快速完成课程评教。请根据您的需求和技术水平选择合适的方案。

## 🚀 方案一：浏览器扩展（推荐）

### 优势
- **即装即用**: 无需安装额外软件
- **用户友好**: 图形化界面，操作简单
- **安全可靠**: 本地执行，无数据泄露风险
- **快捷操作**: 支持快捷键和一键填写

### 文件结构
```
auto_evaluation_extension/
├── manifest.json          # 扩展配置
├── popup.html             # 用户界面
├── popup.js               # 界面逻辑
├── content.js             # 页面操作脚本
├── styles.css             # 样式文件
├── README.md              # 详细说明
└── icon_info.txt          # 图标制作说明
```

### 安装步骤

1. **准备图标文件**（可选）
   - 根据 `icon_info.txt` 创建图标文件
   - 或者暂时跳过，扩展仍可正常工作

2. **安装扩展**
   ```bash
   # 1. 打开Chrome浏览器
   # 2. 访问 chrome://extensions/
   # 3. 启用"开发者模式"
   # 4. 点击"加载已解压的扩展程序"
   # 5. 选择 auto_evaluation_extension 文件夹
   ```

3. **使用扩展**
   - 访问评教页面：`http://210.30.204.138/school/proj/evaluatevl-0/module/task/org/UJUMGRK4kyat8tEaH1z4QN/mytask`
   - 点击扩展图标或按 `Ctrl+Shift+E`
   - 配置评分和评语
   - 点击"自动填写"

### 功能特性

#### 🎛️ 配置选项
- **评分等级**: 1-5分可选（建议选择4-5分）
- **评语内容**: 可自定义评语文本
- **操作延时**: 控制填写速度，避免被检测

#### ⚡ 快捷功能
- **预览模式**: 查看页面中的评教元素
- **一键填写**: 自动填写所有评教项
- **快捷键**: `Ctrl+Shift+E` 快速操作
- **视觉反馈**: 高亮显示操作元素

#### 🔧 智能特性
- **自动识别**: 智能识别评教表单元素
- **分组处理**: 按组处理单选按钮
- **模拟打字**: 自然的文本输入效果
- **错误处理**: 完善的异常处理机制

## 🐍 方案二：Python脚本

### 优势
- **功能强大**: 可以处理复杂的登录和导航
- **高度定制**: 可根据具体需求修改代码
- **批量处理**: 支持多个评教任务
- **详细日志**: 完整的操作日志记录

### 安装依赖
```bash
pip install selenium beautifulsoup4 requests
```

### 下载ChromeDriver
```bash
# 访问 https://chromedriver.chromium.org/
# 下载对应Chrome版本的驱动
# 将驱动文件放到系统PATH中
```

### 使用方法
```bash
python auto_evaluation_tool.py
```

### 配置说明

脚本中的关键配置：

```python
# 评教配置
evaluation_config = {
    'radio_value': '5',  # 评分等级
    'text_content': '老师教学认真负责，课程内容丰富，受益匪浅。',
    'delay_range': (0.5, 2.0)  # 操作延时范围
}

# URL配置
login_url = "http://210.30.204.138/login"  # 登录页面
evaluation_url = "http://210.30.204.138/school/proj/evaluatevl-0/module/task/org/UJUMGRK4kyat8tEaH1z4QN/mytask"
```

## 🔧 自定义和扩展

### 修改评语模板

**浏览器扩展**：
- 直接在弹窗界面修改评语内容
- 配置会自动保存到浏览器存储

**Python脚本**：
```python
evaluation_config = {
    'text_content': '您的自定义评语内容'
}
```

### 调整操作延时

**浏览器扩展**：
- 在界面中调整"操作延时"滑块

**Python脚本**：
```python
evaluation_config = {
    'delay_range': (1.0, 3.0)  # 1-3秒随机延时
}
```

```json
]
```

修改URL配置和元素选择器。

## 🐛 故障排除

### 常见问题

1. **扩展无法加载**
   - 检查开发者模式是否启用
   - 确认文件夹结构完整
   - 查看扩展管理页面的错误信息

2. **无法识别评教页面**
   - 确认在正确的评教页面
   - 检查页面是否完全加载
   - 尝试刷新页面后重试

3. **自动填写失败**
   - 检查网络连接
   - 增加操作延时
   - 查看浏览器控制台的错误信息

4. **Python脚本报错**
   - 检查ChromeDriver版本是否匹配
   - 确认所有依赖已正确安装
   - 查看详细的错误堆栈信息

### 调试方法

**浏览器扩展调试**：
