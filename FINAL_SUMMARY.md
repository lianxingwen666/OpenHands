#### 核心算法
```python
def get_smart_score(strategy):
    """加权随机评分算法"""
    weights = strategies[strategy]['weights']
    random_value = random.random()
    cumulative = 0

    for score, probability in weights.items():
        cumulative += probability
        if random_value <= cumulative:
            return score
```
